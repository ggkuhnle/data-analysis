## 🧰 Requirements (for Local Use)

If you want to run these notebooks locally using Jupyter:

```bash
pip install -r requirements.txt
```

### Main Python Libraries Used:
- `pandas`
- `numpy`
- `matplotlib`
- `seaborn`
- `scipy`
- `statsmodels`
